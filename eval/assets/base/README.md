# eval/assets/base

Place base photo images here for evaluation test cases.

Expected files (matching test_cases.json):
- shirt_white.jpg      — white or light-colored shirt on a person
- shirt_plain.jpg      — plain shirt, front-facing
- wall_plain.jpg       — plain interior wall
- wall_smooth.jpg      — smooth wall, good for texture tests
- mug_plain.jpg        — plain ceramic mug on a table
- notebook_closed.jpg  — closed notebook, cover visible
- notebook_plain.jpg   — plain notebook cover
- poster_frame.jpg     — framed poster on a wall
- cardboard_box.jpg    — cardboard shipping box
- grass_field.jpg      — open grass field

Images should be:
- JPEG or PNG format
- At least 512x512 pixels
- Clear, well-lit, uncluttered
- The target surface should be the dominant element
